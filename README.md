# Fake Job Detection System

A machine learning application that detects fraudulent job postings using Natural Language Processing techniques.

The system analyzes job descriptions and predicts whether a job posting is genuine or potentially fraudulent.

## Features

- Fake and real job classification
- NLP text processing
- TF-IDF feature extraction
- Machine learning prediction
- Confidence score generation
- Interactive prediction interface

## Technologies

- Python
- Pandas
- Scikit-learn
- TF-IDF
- Logistic Regression
- Streamlit
- Machine Learning
